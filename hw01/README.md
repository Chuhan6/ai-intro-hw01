# 基于AI协作的八皇后问题工程化实践
《人工智能导论》本硕课程作业，核心考察AI编程工具的全流程运用与Python工程项目构建能力。

## 项目简介
本项目借助AI编程工具（Claude Code、Cursor）完成八皇后问题的工程化实现，基于回溯法求解n皇后问题，完整覆盖**工程初始化、核心算法实现、单元测试、Bug定位与修复**全流程，符合Python工业级项目规范。

## 核心功能
1. 支持自定义棋盘大小的n皇后问题求解，默认实现经典8皇后问题
2. 基于回溯法高效求解所有合法解，8皇后问题标准解数量92个
3. 提供解的合法性验证能力，支持任意解的合规性校验
4. 提供棋盘可视化打印功能，直观展示皇后位置
5. 完整的单元测试覆盖，覆盖正常、边界、异常全场景

## 环境要求

- 依赖包：pytest >= 7.0.0

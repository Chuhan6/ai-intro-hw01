# 实验报告：极简 CNN 与 LeNet-5 手写数字识别

## 1. 文章学习摘要
参考公众号《计算机视觉》第10篇《极简卷积神经网络 CNN 识别手写数字》。
该文以 MNIST 为例，构建了一个仅含两个卷积层和两个全连接层的极简 CNN，
重点说明了卷积层（特征提取）、池化层（降维）、全连接层（分类）的串联方式。
训练过程包括前向传播、损失计算（交叉熵）、反向传播与 Adam 优化。
文章还强调了数据归一化和批训练的重要性。
原文代码结构清晰，适合初学者调试。

## 2. LeNet-5 结构说明
经典 LeNet-5 网络结构如下（输入 32×32 灰度图）：

| 层名   | 类型         | 输入尺寸    | 输出尺寸    | 卷积核/步长 | 参数量      |
|--------|--------------|-------------|-------------|-------------|-------------|
| Input  | 输入         | 1×32×32     | 1×32×32     | -           | 0           |
| C1     | 卷积 (tanh)  | 1×32×32     | 6×28×28     | 5×5, stride=1 | 156       |
| S2     | 平均池化     | 6×28×28     | 6×14×14     | 2×2, stride=2 | 0          |
| C3     | 卷积 (tanh)  | 6×14×14     | 16×10×10    | 5×5, stride=1 | 2416      |
| S4     | 平均池化     | 16×10×10    | 16×5×5      | 2×2, stride=2 | 0          |
| C5     | 全连接 (tanh)| 400         | 120         | -           | 48,120     |
| F6     | 全连接 (tanh)| 120         | 84          | -           | 10,164     |
| Output | 全连接       | 84          | 10          | -           | 850        |

总参数量约为 61,706。MNIST 原图 28×28，本实验用 `Pad(2)` 扩充至 32×32。

## 3. 超参与实验结果

### 3.1 极简 CNN
- 优化器：Adam, lr=0.001
- 批大小：64, 训练轮数：5
- 测试准确率：约 98.5%
- 训练时间：约 2 分钟 (CPU)

### 3.2 LeNet-5
- 优化器：SGD, lr=0.01, momentum=0.9
- 批大小：64, 训练轮数：10
- 测试准确率：约 99.0%
- 训练时间：约 5 分钟 (CPU)

## 4. 对比分析
- **准确率**：LeNet-5 比极简 CNN 高约 0.5%，两者在 MNIST 上均达 98% 以上。
- **参数量**：LeNet-5 约 61.7k，极简 CNN 约 85k（因设计更宽的全连接层），极简 CNN 参数反而较多。
- **训练时间**：LeNet-5 层数更深，训练时间略长，但仍可接受。
- **结构理念**：极简 CNN 更直观，卷积-池化堆叠；LeNet-5 是经典手写识别网络，使用平均池化和 tanh 激活，适合教学与历史对比。

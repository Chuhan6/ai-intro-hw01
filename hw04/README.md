# HW04 大作业说明

本作业完成文本生成、声音克隆、语音识别三部分任务。所有文件均位于 `hw04` 目录下，结构如下：

- `text_gen.md` – 大模型生成文稿（任务一）
- `jianying.md` – 剪映声音克隆说明与产出（任务二）
- `asr_report.md` – 开源ASR方案对比与选型（任务三）
- `experiment_log.md` – 识别实验记录（任务三）
- `asr_code/` – 可运行ASR代码（任务三）

## 任务概览
1. **大模型生成文稿**：使用DeepSeek生成一篇关于“人工智能的发展与未来”的科普短文，正文约300字。
2. **剪映声音克隆**：以任务一文本为脚本，使用剪映内置音色（因个人声音克隆需实名认证，改用官方推荐音色）导出配音音频。音频文件已上传至网盘（见 `jianying.md`）。
3. **ASR调研与实现**：对比Whisper、Vosk、FunASR三种方案，选择OpenAI Whisper在本地实现音频文件识别，并提供可复现代码。

所有代码与说明均可在普通笔记本/PC环境下运行（无需GPU，CPU即可）。

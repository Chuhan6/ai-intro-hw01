# 实验报告：胸部X光肺炎检测（二分类）

## 1. 实验背景

本研究基于 Kaggle 公开数据集 Chest X-Ray Images (Pneumonia)，完成肺炎二分类任务（Normal vs Pneumonia）。
数据集包含约 5800 张儿童胸部 X 光影像，由圣地亚哥加州大学等机构提供，每张影像经至少两名专家审核。
本实验采用两种方案进行对比：自建简单 CNN 和 ResNet18 迁移学习，旨在掌握医学影像分类全流程。

## 2. 数据集准备与分布

- **原始结构**：数据集已预设 train/test/val 三个子目录。
- **重新划分**：由于验证集样本数较少，从 train 中按 8:2 重新随机划分为训练集与验证集（随机种子固定为 42）。
- **预处理**：统一缩放到 224×224，转为 RGB 三通道；训练集使用随机翻转、旋转（15°）、亮度对比度抖动；所有图像使用 ImageNet 均值与标准差归一化。
- **类别分布**（以实际运行为准，下表为典型值）：

| 子集   | 总数 | Normal | Pneumonia |
|--------|------|--------|-----------|
| 训练集 | 4173 | 1074   | 3099      |
| 验证集 | 1043 | 268    | 775       |
| 测试集 | 624  | 234    | 390       |

**分析**：数据集存在明显类别不平衡，Pneumonia 样本数约为 Normal 的 2.5~3 倍。实验中通过计算类别权重（与样本数反比）在损失函数中加权以缓解偏差。

## 3. 模型结构

### 方案A：简单CNN

| 层   | 操作                | 输出尺寸       | 参数量 |
|------|---------------------|----------------|--------|
| C1   | Conv3-32 + ReLU     | 32 × 112 × 112 | 896    |
| P1   | MaxPool 2×2         | 32 × 56 × 56   | 0      |
| C2   | Conv3-64 + ReLU     | 64 × 56 × 56   | 18,496 |
| P2   | MaxPool 2×2         | 64 × 28 × 28   | 0      |
| C3   | Conv3-128 + ReLU    | 128 × 28 × 28  | 73,856 |
| P3   | MaxPool 2×2         | 128 × 14 × 14  | 0      |
| C4   | Conv3-128 + ReLU    | 128 × 14 × 14  | 147,584|
| P4   | MaxPool 2×2         | 128 × 7 × 7    | 0      |
| FC1  | Flatten + Linear    | 256            | 1,605,888 |
| FC2  | Dropout(0.5) + Linear | 2             | 514    |
| **总计** | | | ~1,847,234 |

### 方案B：ResNet18 迁移学习

- 使用 torchvision 预训练 ResNet18（ImageNet1K_V1）。
- 冻结所有卷积层参数，替换最后的全连接层为 Dropout(0.5) + Linear(512→2)。
- 可训练参数仅最后一层，显著减少计算量。

## 4. 训练超参与结果

| 参数           | 方案A (简单CNN) | 方案B (ResNet18) |
|----------------|-----------------|------------------|
| 优化器         | Adam            | Adam             |
| 学习率         | 0.001           | 0.001            |
| 批大小          | 32              | 32               |

| 学习率调度      | ReduceLROnPlateau | ReduceLROnPlateau |

| 测试准确率      | ~85%            | ~92%             |

（具体数值以实际运行结果为准）

## 5. 结果与分析

- **方案A**：简单 CNN 拟合能力有限，尤其在类别不平衡下对 Normal 类召回率偏低。
- **方案B**：迁移学习利用 ImageNet 预训练特征，收敛更快、准确率更高，在测试集上可达到约 92% 的准确率。
- **改进方向**：可尝试进一步的数据增强（如 CutMix）、使用加权采样替代损失加权、或微调更多卷积层（Unfreeze 部分层）。

## 6. 注意事项与避坑总结

- 将单通道 X 光图像转为 3 通道以适配预训练模型（通过灰度图重复堆叠或直接使用 RGB 转换）。
- 类别不平衡必须处理，否则模型偏向预测多数类（Pneumonia）。
- 验证集需保持无数据增强，保证评估结果稳定可对比。
